How to submit data

  • The file should be encoded as UTF-8
  • The file should be saved in CSV format with TAB seperators between fields
  • When there is no data available for a field it should be marked using na
  • True and False fields should be marked as TRUE and FALSE
  • The file may be created using the Fantallometrik software available under the Software section of this website
After your data is submitted, it will be professionally reviewed before being added to the GlobAllomeTree database.
