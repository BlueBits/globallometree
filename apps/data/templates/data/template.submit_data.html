How to submit data

  • The file should be encoded as {{ export_encoding }}
  • The file should be a tab-separated values file saved with a .txt extension and use TAB field delimiters
  • When there is no data available for a field it should be marked using None
  • True and False fields should be marked as True and False
  • The file may be created using the Fantallometrik software available under the Software section of this website
  • Numbers and Decimals should be entered according to the French Numeric System with a comma "," used for the decimal seperator and the dot "." optionally used as the thousands seperator
