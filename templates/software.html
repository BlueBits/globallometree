1 Compare allometric equations

This module allows the comparison of existing allometric equations for a given geographical location (a site, an ecological zone, a country, a continent). The user can select the ecological classification from five different sources (FAO, Udvardy, WWF, Holdridge, and Bailey). The results obtained from the analysis can be downloaded by the users.

2 Assess volume, biomass and carbon stocks

This module allows the calculation of volume, biomass and carbon stocks, the calculation of descriptive statistics (error, confidence interval, median etc.). and comparison of data obtained according to the default values from the scientific literature. Moreover, it is a robust tool to assess the variable of interest and achieve the quality control and assurance.

3 Insert new equations

This module allows the entry of new allometric equations and enrich the database. New allometric equations are automatically inserted into the database of the user and can be submitted to Fantallometrik after his approval.

4 Share your results

This module allows the user to submit user data (allometric equation, inventory data). The data submitted is protected by the license by-nc-sa.

Presentation

The estimation of stem volume and tree biomass is needed for both sustainable planning of forest resources and for studies on the energy and nutri¬ents flows in ecosystems. Planners at the strategic and operational levels have strongly emphasised the need for accurate estimates of stem volume, and biomass as an energy source. In addition, the United Nations Framework Convention on Climate Change recognises the importance of forest carbon sink and the need to monitor, pre¬serve and enhance terrestrial carbon stocks, since changes in the forest carbon stock influence the atmospheric CO2 concentration.

Terrestrial biotic carbon stocks and stock changes are difficult to assess and most current estimates are subject to considerable uncertainty particularly in developing countries where the data are lacking and the structure of the ecosystems is more complex. The reliability of the current estimates of the forest carbon stock and the understanding of ecosystem carbon dynamics can be improved by applying existing knowledge on the allometry of trees that is available in the form of biomass and volume equations. The volume and biomass equations can be applied directly to tree level inventory data using wood density data and biomass expansion factors (BEFs) to make the estimates comparable.

In order to decrease the uncertainty related to the use of inappropriate allometric models, it is preferable to use local specific allometric equations rather than generalized ones. For that purpose, the software allows you accessing the allometric equation databases and select those that are the most adapted to your purpose. The calculation steps are the following ones:

  1. identify the allometric equations that are available,
  2. insert new equations that are not already inserted in the database,
  3. estimate volume, biomass and carbon stocks with the associated statistical data and compare with default value, and
  4. if you wish, you can submit your data to enrich the database and contribute to the “fantallometrik” community.

Instructions

FANTALLOMETRIK Software (c) is a free software and works only under windows operating systems. It is required to accept a license agreement before downloading.

  • FANTALLOMETRIK V1 release:
    • Default allometric equation database containing 850 allometric equations for sub-Saharan Africa
    • Default wood density database containing 2650 wood density values
    • 4 Modules [comparison, calculator, data insertion, submission]
  • Installing FANTALLOEMTRIKV1 on your machine
    1. Attempt to verify the MATLAB Component Runtime (MCR) exists on your system: the MCR usually resides in C:\Program Files\MATLAB\MATLAB Component Runtime\v76
    2. If this directory exists, check that it is included in the environment variable 'PATH', otherwise, add the MCR directory to the environment variable
      • Either by opening a command prompt and issuing the DOS command: set PATH=C:\Program Files\MATLAB\MATLAB Component Runtime\v76;%PATH%
      • Or, add C:\Program Files\MATLAB\MATLAB Component Runtime\v76 to the PATH environment variable by doing the following:
        1. Select the My Computer icon on your desktop.
        2. Right-click the icon and select Properties from the menu.
        3. Select the Advanced tab.
        4. Click Environment Variables. Your environment variables are listed.
    3. If the directory does not exist, download and execute MCRInstaller_R2010A.exe (self-extracting MATLAB Component Runtime library archive) . To achieve this, you need to ask your administrator system.
    4. then, to execute FANTALLOMETRIK, click twice on Fantallometrik.exe FANTALLOMETRIK License Agreement
